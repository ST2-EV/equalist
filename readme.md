# Equalist
A group playlist creator that takes your and your friends spotify history into account.





### Run
##### backend: 
```cd backend``` \
best setup a venv now \
```pip install -r requirements.txt``` \
```python3 -m uvicorn api.main:app --host 0.0.0.0 --port 8000```


##### frontend: 
```cd frontend```
```npm run dev```
